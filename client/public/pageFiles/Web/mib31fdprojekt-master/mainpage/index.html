Was sind eigentlich HTML & CSS ?

HTML

HTML (Hypertext Markup Language) bildet das Grundgerüst einer jeden Webseite. Es handelt sich um eine Auszeichnungssprache, die verwendet wird, um den Inhalt einer Webseite zu strukturieren und zu organisieren. HTML definiert die verschiedenen Elemente einer Seite, wie Überschriften, Absätze, Listen, Bilder und Links. Diese Elemente werden durch Tags markiert, die den Browsern mitteilen, wie die Seite dargestellt werden soll.
Durch die semantische Struktur von HTML wird nicht nur die visuelle Darstellung, sondern auch die Bedeutung einzelner Abschnitte einer Webseite klar und verständlich für sowohl Browser als auch Entwickler.

CSS

CSS (Cascading Style Sheets) hingegen ist eine Designsprache, die es ermöglicht, das Aussehen und Layout einer Webseite zu gestalten. Mit CSS können Entwickler das Erscheinungsbild von HTML-Elementen verändern, indem sie verschiedene Stileigenschaften wie Farben, Schriftarten, Abstände und Positionen definieren. Durch die Trennung von Inhalt (HTML) und Design (CSS) wird die Wartung und Anpassung von Webseiten erleichtert. CSS erlaubt es, ein konsistentes und ansprechendes Design über alle Seiten einer Webseite hinweg zu erstellen, und ermöglicht somit eine flexible Anpassung des Erscheinungsbilds entsprechend den gestalterischen Anforderungen.

Ziele

  • Spielerische Grundlagenvermittlung

    Entdecke die Grundlagen von HTML & CSS spielerisch in "Towers of Tags". Hier lernst du ohne Stress und durch praxisfokussierte Aufgaben, begleitet von kurzen theoretischen Einführungstexten.

  • Präzises Wiederholen von Inhalten

    Spezifisch auf Themen ausgerichtete Levels ermöglichen gezieltes Wiederholen von Schwachstellen, um sicherzustellen, dass die erlernten Inhalte nachhaltig verstanden werden.

Tutorial

1

Schreibe HTML Code in das Coding Panel um Tower im Spiel zusetzen und deine Gegner aufzuhalten

2
Verschiedene HTML Tags erschaffen verschiedene Tower-Arten. Lerne welche Kombis dich zum Sieg führen und lerne dabei unterschwellig wie verschiedene HTML Elemente einzusetzen sind!
3
Gib den Towern Eigenschaften & Fähigkeiten mit CSS! Position, Größe, Farbe, den Möglichkeiten sind keine Grezen gesetzt!
4
Drück auf Play und probiere aus ob deine Kombination alle ankommenden Gegner besiegen kann oder ob du dich nochnal dransetzen musst!
