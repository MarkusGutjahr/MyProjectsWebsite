Im Bereich des Webdesigns wird, wie der Name schon sagt, sich auf das Design fokussiert. Es wird nicht nur eine Webseite programmiert, vielmehr wird es um die kreative Gestaltung einer Webseite gehen.

Die Wahlpflichtmodule konzentrieren sich neben der Verwaltung von Dokumenten hauptsächlich auf die Erstellung und Bedienung von touchbasierten Systemen. Im Fokus werden anhand neuester Technologien Web-Frameworks gestaltet und moderne Webanwendungen entwickelt

Welche neuen Technologien gibt es und wie werden sie angewendet? Ein Beispiel sind Lernapps oder moderne Lernangebote, die mittlerweile undenkbar sind. Hier wird Technologie und Lernen miteinander verbunden und praxisnah erlebt.

HTML dient zur Strukturierung von Inhalten, wie beispielsweise Texte und Bilder. Die dabei entstehenden Dokumente sind die Grundlage des WWW und werden von Webbrowsern genutzt, um die Inhalte aller Seiten darzustellen.

CSS ist eine Formatierungssprache, welche dazu dient, Dokumente gestalterisch aufzuwerten. Dazu werden verschiedenste Style Elemente genutzt, um allen Wünschen der Nutzer nachzukommen. CSS allein kann allerdings keine Inhalte darstellen, es kann Inhalte lediglich verändern.

Java Script dient die Inhalte der HTML-Dokumente dynamisch darzustellen. Dabei können beispielsweise durch Benutzerinteraktionen neue Inhalte erscheinen oder auch angezeigte Elemente durch den Nutzer, oder auch von allein bewegt werden. JS allein ist nicht dazu gedacht, Inhalte darzustellen, es wird genutzt, um die erstellen Inhalte dynamisch für den Nutzer anzuzeigen.

Die Kombination aus HTML und CSS sorgt für eine visuell ansprechendere Gestaltung der Inhalte.

Die Kombination aus HTML und JS sorgt für eine dynamische Webseite, welche allerdings nicht visuell ansprechend für den Nutzer gestaltet ist.

Die Kombination von CSS und JS ist nicht dazu gedacht, Inhalte darzustellen, sie werden genutzt, um die erstellen Inhalte sowohl visuell ansprechend, als auch dynamisch für den Nutzer anzuzeigen.

Die Kombination von HTML, CSS und JS ergeben visuell ansprechende und dynamische Inhalte, die nach den Wünschen der Nutzer eingerichtet werden können.
